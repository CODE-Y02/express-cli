# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | ✅        |
| next    | ✅        |
| < 1.0   | ❌        |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.
Instead, contact the maintainer privately:

- **LinkedIn**: [Yatharth Lakhate](https://www.linkedin.com/in/yatharth-lakhate/)
- **X (Twitter)**: [@Yatharth_L](https://x.com/Yatharth_L)

We will respond promptly to acknowledge the report and will work quickly with you to issue a patch. We'll ensure the vulnerability is addressed securely before any public disclosure occurs.
