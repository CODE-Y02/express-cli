# Getting Started

## Quick Start

The fastest way to create a new Express project is to use `npx`:

```bash
npx create-express-forge
```

## Features

- **TypeScript First**: Full type safety from the start.
- **Architecture**: Choice between Modular (recommended) and MVC.
- **Prisma/Sequelize**: Integrated ORM support.
- **Testing**: Pre-configured Vitest or Jest.
- **Docker**: Production-ready Docker setups.
