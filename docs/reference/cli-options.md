# CLI Reference

You can customize your project creation using these flags:

| Flag | Description |
|------|-------------|
| `--help` | Show help |
| `--version` | Show version |
| `--pattern` | Choose architecture (modular/mvc) |
| `--orm` | Choose ORM (prisma/sequelize) |
| `--db` | Choose database (postgres/mysql/sqlite) |
| `--logger` | Choose logger (winston/pino) |
| `--test` | Choose framework (vitest/jest) |
| `--docker` | Add docker files (true/false) |
| `--install` | Install dependencies (true/false) |
